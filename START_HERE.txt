📊 DATASET INFORMATION

Dataset: olympic_countries_efficiency.csv
├─ Records: 250+ Olympic participation records
├─ Time Period: 1992-2016 Summer Olympics
├─ Countries: 100+ nations represented
├─ Features: 18 columns covering:
│  ├─ Socioeconomic: population, GDP per capita, income group
│  ├─ Participation: athletes, sports, events
│  ├─ Performance: medals (Gold, Silver, Bronze)
│  ├─ Demographics: female athlete percentage
│  └─ Historical: previous medal performance
├─ Target: total_medals, medals_per_athlete
└─ Quality: High (minimal missing values)
